Outils de gestion de projet 

L’équipe de Canada-Translate a réuni dans cette section une liste exhaustive des outils de gestion disponibles gratuitement en ligne qui peuvent s’avérer très utiles pour le traducteur professionnel.

Les divers outils de gestion de projet utiles pour le traducteur :

  • Gestan
    Avec Gestan, vous suivez vos clients et vos contacts, vos devis, factures et commandes, vos temps de travaux, vos produits et vos stocks, vos tâches, votre comptabilité...
  • Toodledo
    An easy to use, online to-do list. Get organized, stay motivated, and be more productive.
  • Doodle
  • Dropbox
  • Read It Later
  • Phras.In
  • Corrige-Moi
  • iDispo
  • Count Anything
    Word/character counts are an essential part of the work of translators, writers, and others who need to bill based word count. There are several word-count utilities around, some of them free, but I wrote this one because the other tools didn't count Asian characters and/or didn't support some MS Office formats. Count Anything counts the words and characters in a variety of file formats. While it doesn't quite count anything, it supports the following file types: Word (.doc, .rtf), Excel (.xls, .csv), PowerPoint (.ppt), Writer (.odt), Impress (.odp), Calc (.ods), HTML, XML, Text, PDF. It counts words in text boxes and many other embedded objects. It counts the numbers of words, characters (with and without spaces), Asian characters, and non-Asian words.
